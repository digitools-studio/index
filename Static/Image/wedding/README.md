# Wedding Invitation 圖片配置

此目錄專門存放婚禮邀請函的背景圖片。

## 圖片命名規範

為配合 wedding_invitation.html 的6個頁面，請按以下方式命名圖片檔案：

### 必要圖片檔案

1. **slide-1.jpg** - 首頁封面
   - 建議尺寸：1920x1080 或更高
   - 用途：顯示婚禮主標題、新人名字、日期及地點資訊
   - 建議風格：浪漫、優雅的婚禮場景

2. **slide-2.jpg** - 邀請詞頁面
   - 建議尺寸：1920x1080 或更高
   - 用途：顯示邀請詞及婚禮日期
   - 建議風格：溫馨、文藝的畫面

3. **slide-3.jpg** - 純照片展示
   - 建議尺寸：1920x1080 或更高
   - 用途：純圖片展示（無文字覆蓋）
   - 建議風格：新人合照或婚紗照

4. **slide-4.jpg** - 誓言頁面
   - 建議尺寸：1920x1080 或更高
   - 用途：顯示婚禮誓言文字
   - 建議風格：浪漫、莊重的背景

5. **slide-5.jpg** - 祝福詞頁面
   - 建議尺寸：1920x1080 或更高
   - 用途：顯示祝福語
   - 建議風格：明亮、唯美的畫面

6. **slide-6.jpg** - 詳細資訊頁面
   - 建議尺寸：1920x1080 或更高
   - 用途：顯示詳細的婚禮時間、地點及地圖導航
   - 建議風格：清晰、易讀的背景

## 圖片格式建議

- **格式**：JPG（壓縮後體積小）或 WebP（現代瀏覽器支援，體積更小）
- **品質**：70-85% 壓縮率，平衡畫質與檔案大小
- **色調**：建議統一色調，保持視覺一致性
- **文字可讀性**：確保背景不會過於複雜，影響白色文字的閱讀

## Fallback 機制

- 若本地圖片載入失敗，系統會自動使用 Unsplash 的備用圖片
- 備用圖片可在 wedding_invitation.html 的 `data-bg-fallback` 屬性中設定

## 優化建議

1. 使用圖片壓縮工具（如 TinyPNG、Squoosh）減小檔案大小
2. 考慮提供多種解析度版本（如 @1x, @2x）適配不同螢幕
3. 移動端可考慮使用較小的圖片版本
4. 使用 WebP 格式可節省約 30% 的檔案大小

## 圖片來源建議

- 專業婚攝照片
- 婚紗攝影作品
- 精選的高品質免費圖庫（如 Unsplash、Pexels）

---

**注意**：請確保所有使用的圖片都有適當的使用權限。
