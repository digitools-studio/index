常見問題

SignFlow PDF 是什麼工具？
SignFlow PDF 是免費的線上 PDF 電子簽名工具，支援手寫簽名（自動去背）、電子印章 / 章圖上傳蓋章、文字標記及框線繪製。所有操作在瀏覽器本機完成，無需安裝任何軟體，完成後立即匯出標準 PDF 文件。
PDF 文件會上傳到伺服器嗎？資料安全嗎？
不會。所有 PDF 文件處理與簽名合成完全在您的瀏覽器本機進行，文件內容不會上傳至任何伺服器，保障商業機密與個人隱私。草稿透過瀏覽器 IndexedDB 儲存於本機裝置。
可以在 PDF 上蓋電子章或公司印章嗎？
可以。上傳 PNG 或 SVG 格式的章圖後，可拖曳到 PDF 任意位置，支援縮放、旋轉與透明度調整。常用印章可儲存至資產庫，下次一鍵套用。
完全免費嗎？有功能限制嗎？
基本功能完全免費，Google 登入後即可使用手寫簽名、印章蓋章、文字標記、草稿保存與 PDF 匯出，無頁數限制。另有企業客製開發服務提供洽詢。
匯出的 PDF 相容 Adobe Acrobat、macOS Preview 嗎？
完全相容。SignFlow PDF 以標準 PDF 格式匯出，相容 Adobe Acrobat、macOS Preview、Windows Edge、Chrome、Foxit 等主流 PDF 閱讀環境。
