隱私權政策

最後更新：2024 年 12 月

前言

Digitools Studio（以下簡稱「本公司」）重視您的隱私權與個人資訊保護。本政策說明我們如何收集、使用、保護及處理您的個人資料。建議您仔細閱讀本政策，若有疑問請聯繫我們。

1. 我們收集的個人資料

本公司透過以下方式收集您的個人資料：

聯絡表單（contact.html）

  • 姓名、電子郵件、電話（您主動提供）
  • 公司名稱、服務需求、預算範圍（可選欄位）
  • 聯絡表單經由 Formspree 服務傳遞

自動收集的資料

  • IP 位址、裝置類型、作業系統（Google Analytics）
  • 頁面瀏覽紀錄、停留時間、點擊行為
  • 來源網址、搜尋關鍵字

2. 個人資料的使用目的

📧

服務聯繫與回覆

回應您的查詢、提供報價或服務資訊

📊

網站分析與改進

透過 Google Analytics 分析訪客行為，優化網站使用體驗

📈

成效追蹤與 KPI 分析

追蹤用戶與服務頁、聯絡表單的互動，優化轉換漏斗

✉️

行銷與溝通

若您同意，我們可能透過電子郵件發送最新服務資訊（可隨時取消訂閱）

3. 資料安全保護措施

本公司採取以下措施保護您的個人資料：

  • 使用加密技術（HTTPS）傳輸敏感資訊
  • 限制內部員工對個人資料的存取權限
  • 定期檢查並更新資安防護措施
  • 遵守相關個人資料保護法規

4. 第三方服務提供商

本公司與以下第三方服務商共享部分資料：

Formspree

用途：聯絡表單提交與資料傳遞

資料：姓名、電子郵件、公司名稱、服務需求

Formspree 隱私政策 →

Google Analytics & Google Tag Manager

用途：網站流量分析、用戶行為追蹤

資料：匿名化 ID、頁面瀏覽、事件追蹤

Google 隱私政策 →

GitHub Pages

用途：網站託管

資料：伺服器日誌、訪客 IP（自動收集）

GitHub 隱私聲明 →

5. Cookie 與追蹤技術

本網站使用 Cookie 與類似技術追蹤用戶行為，幫助我們改進服務：

  • Google Analytics Cookie：用於分析網站流量與用戶行為
  • Google Tag Manager：追蹤特定事件（服務頁 CTR、表單漏斗、停留時間）
  • 您可透過瀏覽器設定管理或禁用 Cookie，但可能影響網站功能

6. 您的個人資料權利

根據個人資料保護法（GDPR / CCPA 或中華民國個資法），您擁有以下權利：

✓

查閱權

了解我們保有您的哪些個人資料

✓

更正權

要求更正不正確的個人資料

✓

刪除權

要求刪除您的個人資料（部分情況下適用）

✓

反對權

反對我們對您的個人資料進行特定處理

✓

資料可轉移權

要求以結構化、通用格式取得您的資料

若要行使上述權利，請聯繫我們： hello@digitools.studio

7. 資料保留期限

聯絡表單資料

保留至服務完成或客戶明確要求刪除，通常不超過 2 年

Google Analytics 資料

預設保留 26 個月；超過期限自動刪除

行銷郵件

保留期間直至您取消訂閱

8. 政策變更

本政策可能因應法律規定或業務需要而更新。重大變更時，我們將透過電子郵件或網站公告通知您。持續使用本網站表示同意最新政策。
